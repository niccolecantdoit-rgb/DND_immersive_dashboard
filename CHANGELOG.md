# 更新日志


## [1.9.0] - 2026-02-07

### ✨ 新增功能
- **完整的主题自定义功能**：
    - **多套内置主题**：新增了经典 DND (Classic)、赛博霓虹 (Cyber Neon)、暗黑哥特 (Gothic Horror)、精灵之森 (Elven Forest) 等多套高品质主题预设。
    - **深度样式定制**：不仅是颜色，现在支持对边框、圆角、卡片形状、纹理特效、字体等进行全方位的视觉定制。
    - **动态背景特效**：主题现在包含粒子 (Particles)、雷达 (Radar)、脉动光效 (Pulse) 等多种动态背景，沉浸感拉满。
    - **风格化动画**：不同主题拥有专属的交互动画，例如霓虹闪烁、血液流动、藤蔓生长等。
- **UI 渲染引擎升级**：
    - **CSS 变量系统重构**：全面重构了底层的 CSS 变量系统，引入了形态变量 (Morphology) 和交互状态变量，支持更复杂的视觉表现。
    - **动态样式注入**：优化了样式加载机制，支持运行时动态切换主题而无需刷新页面。

### 🎨 界面与动画
- **视觉反馈增强**：
    - **交互动画**：新增了列表项级联入场、HUD 弹窗微动效、点击波纹反馈等细腻的交互动画。
    - **状态感知**：当前行动者的 Token 在小地图上会显示独特的呼吸光环和动态边框。
    - **流光特效**：状态条 (HP/XP) 和行动按钮在特定状态下会显示流光溢彩的动态效果。
- **宽屏与移动端适配**：
    - **响应式布局优化**：针对宽屏和移动端分别进行了布局优化，侧边栏、HUD 和详情卡片在不同设备上都能完美展示。
    - **触摸体验改进**：移动端增加了悬浮按钮的触摸优化和防误触处理。

### ⚡ 性能优化
- **样式按需加载**：通过新的 `ThemeManager` 和 `StylePresets` 模块，实现了样式的按需计算和应用，减少了不必要的渲染开销。

## [1.8.0] - 2026-02-06

### ✨ 新增功能
- **NPC 战斗数据扩展**：
    - **数据结构升级**：`sheet_NPC_Registry` 表新增了等级、HP、AC、主要技能、随身物品等 5 个关键战斗字段。
    - **全屏图鉴增强**：人物图鉴现在直接显示 NPC 的血条、护甲等级 (AC) 和职业等级，详情页支持查看技能和物品标签。
- **MiniHUD NPC 列表**：
    - **快捷访问**：在 HUD 底部新增了 NPC 列表按钮 (👥)，可快速查看当前场景的所有 NPC。
    - **实时筛选**：支持按姓名搜索或按状态（在场/死亡等）筛选 NPC。
    - **详情弹窗**：点击列表中的 NPC 可弹出详细信息卡片，包含战斗数据、外貌描述和关键经历。
- **表格管理器模块**：
    - **可视化数据编辑**：全新的表格管理面板，可直观查看和编辑所有数据表（如角色、NPC、物品、技能等）。
    - **卡片式视图**：每条记录以精美卡片形式展示，支持单列/双列布局切换。
    - **新增/删除记录**：一键添加新记录或删除现有记录，带确认对话框防止误操作。
    - **实时搜索**：支持关键词实时过滤，快速定位目标数据。
    - **智能标题**：自动识别姓名、名称等字段作为卡片标题，提升可读性。

### 🎨 界面与动画
- **统一弹窗样式**：重构了模态框 (Modal) 系统，统一了全屏界面和 MiniHUD 的弹窗风格，拥有统一的金色边框、高亮标题和关闭按钮。
- **视觉反馈**：NPC 卡片的血条会根据血量比例显示不同颜色（绿/黄/红）。
- **全组件美化**：对插件内所有组件进行了全面的视觉美化，统一了设计语言，提升整体质感。
- **加强动画效果**：优化并增强了各处的动画交互效果，操作反馈更加细腻流畅。

### 🐛 修复
- **弹窗关闭失效**：修复了全屏模式下，NPC 详情弹窗无法通过关闭按钮关闭的问题。
- **样式缺失**：补全了 `.dnd-modal` 等相关 CSS 样式，确保弹窗在各种分辨率下都能正确居中显示。
- **确认对话框定位**：修复了在窄屏或特殊环境下，删除确认对话框显示在屏幕顶部而非居中的问题。
- **对话框层级**：修复了确认对话框被 MiniHUD 遮挡的问题，现在对话框始终显示在最顶层。
- **新增记录显示**：修复了点击新增按钮后，新记录无法显示的问题（之前空行会被跳过渲染）。


## [1.7.0] - 2026-02-05

### ✨ 新增功能
- **沉浸式背景系统**：
    - **动态特效**：新增齿轮 (Gears)、水波纹 (Ripples)、粒子 (Particles) 等多种动态背景特效。
    - **滚动纹理**：新增条纹 (Stripes)、点阵 (Dots)、六边形 (Hex) 等无缝滚动图案。
    - **自定义配置**：设置面板新增背景选项，支持实时切换特效与纹理。

### 🎨 界面与特效
- **视觉体验升级**：
    - **可视性优化**：全面调整了特效的透明度与对比度，确保背景不会干扰前景文字阅读。
    - **移除干扰项**：根据社区反馈，移除了视觉干扰较强的"数字雨"特效。
- **高清矢量图标**：全面移除 Emoji 图标，替换为 FontAwesome SVG 矢量图标。图标现在支持自定义颜色、阴影，并且在任何分辨率下都保持清晰。
- **动态交互特效**：
    - **悬停光晕**：鼠标悬停在按钮、物品或技能上时，图标会产生与主题色匹配的柔和光晕和轻微放大效果。
    - **状态感知动画**：
        - ⬆️ **升级提醒**：当队友可升级时，队伍栏的箭头图标会自动**跳动**以提示玩家。
        - 📌 **任务提示**：进行中的任务图标会显示**金色呼吸光晕**，吸引注意力。
- **主题自适应**：所有 SVG 图标都会自动继承当前主题的文本颜色（如高亮色、警告色），保持视觉风格高度统一。

### ⚡ 性能优化
- **SVG Tree Shaking**：配置了 Webpack 按需打包，仅将使用到的 50+ 个图标代码打包进插件，无需从外部 CDN 下载字体文件，实现了**完全离线可用**且体积最小化。

### 🧠 智能体验优化
- **装备特效不丢失**：修复了装备在更新描述时可能丢失魔法属性的问题。现在你的神器特效会被妥善保管，不会被剧情文本冲掉。
- **技能录入更精准**：
    - **完美支持自制技能**：如果你在玩魔改或自制规则，AI 现在会优先保留你原本的技能描述，而不会强制套用 DND 标准书。
    - **拒绝偷工减料**：AI 现在会老老实实地完整摘录长文本技能的效果，不再会为了省事而简写或略过关键细节。

### 🐛 修复
- **战斗地图点击定位**：修复了在战斗模式下使用瞄准功能时，点击位置与实际选中位置不一致的问题。现在无论地图如何缩放或拖动，点击都能精准选中目标格子。
- **缩放即时生效**：修复了调整 UI 缩放后，地图点击定位需要重新加载脚本才能正常工作的问题。现在调整缩放后立即生效，无需刷新。

## [1.6.0] - 2026-02-05

### ✨ 新增功能
- **角色升级系统上线**：现在支持从当前状态平滑升级到下一等级，包括生命值、特性、法术和属性的提升。
- **世界书自适应升级**：升级向导现在会优先读取并参考启用的世界书（World Info）。如果你的世界有特殊的魔法体系、种族或职业，AI 会根据这些设定生成自定义的升级选项，而不仅仅局限于 DND 5E 规则。
- **云端配置同步**：插件的配置设置现在通过 `TavernSettingsSync` 与酒馆云端同步，切换设备也能保留偏好。

### ⚡ 优化
- **智能去重**：修复了多次升级或创建角色时，技能和特性出现大量重复的问题。现在系统会自动扫描并合并同名条目，保持面板整洁。
- **自动描述更新**：当 AI 生成了已有特性的新描述（例如根据等级提升了数值），系统会自动更新现有条目的描述，而不是创建新条目。
- **更积极的经验结算**：优化了填表模板的提示词，现在 AI 会在战斗胜利、任务完成或解开谜题后，更主动地在当前轮次结算并增加经验值 (XP)，无需玩家反复催促。

### 🐛 修复
- **技能重复 Bug**：彻底修复了角色面板中显示多个相同技能/特性的问题。
- **描述文本过时**：修复了某些随等级成长的特性（如回风转元）描述未更新的问题。


## [1.3.1] - 2026-01-31

### 修复
- **战斗地图点击失效**：修复了在战斗模式下，瞄准绿圈范围内无法点击地图移动或选择敌人的严重 Bug。原因是地图缩放功能 (`bindMapZoom`) 中的 `e.preventDefault()` 阻止了浏览器生成 click 事件。现在通过检测瞄准模式 (`_targetingMode.active`) 或点击 Token (`.dnd-minimap-token`) 时禁用拖拽功能，让原生 click 事件正常传播。


## [1.3.0] - 2026-01-20

### 新增
- **队伍导入/导出**：新增功能，支持将当前队伍数据导出为 JSON 并导入。
- **战斗角色操控**：增加了在战斗中在主要 PC 和队友之间切换操控的能力。
- **资源追踪**：实现了战斗期间法术位和职业资源消耗的实时追踪。
- **物品详情增强**：物品卡片现在显示包括伤害、特性、稀有度、重量和价值在内的详细信息。
- **手动数据更新**：在 HUD 底部添加了刷新按钮，用于手动触发数据同步。
- **FVTT 支持**：添加了从 Foundry VTT 导入数据的支持。
- **角色创建交互**：在 AI 角色创建向导中新增了点击选项功能，用户现在可以直接点击 AI 提供的选项按钮进行回复，无需手动输入。
- **资源统计重置**：在战斗面板的资源消耗统计栏中添加了手动重置按钮 (🔄)，允许用户随时清零当前的消耗统计。
- **动作经济系统**：新增了回合内动作资源的追踪系统。现在会限制每回合使用的动作（Action）、附赠动作（Bonus Action）和反应（Reaction）数量，以及移动距离（默认30尺）。
- **动作资源调整**：战斗 HUD 中的动作状态栏现在支持点击交互。点击对应的动作类型可以手动增加资源，以支持"动作如潮"或"急行"等特殊能力。
- **技能自动化**：现在使用"疾走" (Dash) 会自动增加本回合移动力，使用"动作如潮" (Action Surge) 会自动增加额外的标准动作，无需手动调整。

### 修复
- **HUD 渲染**：修复了 HUD 内容选择器不正确导致 UI 渲染失败的问题。
- **角色操控一致性**：修复了在切换操控角色时，战斗面板的技能列表、移动逻辑和瞄准范围未正确跟随当前角色的问题。现在所有战斗操作都严格基于当前操控的角色。
- **行动叙事**：修正了行动执行时的文本输出前缀，现在会明确显示 "轮到[角色名]的回合时"，而不是统一显示 "轮到我的回合时"。
- **资源显示修复**：修复了切换角色时资源消耗统计未正确重置，导致显示前一角色资源差值的 Bug。
- **数据保存**：修复了在角色创建过程中，专长（Feats）和技能（Skills）数据无法写入数据库的问题。现在查找逻辑更加健壮，支持无前缀的表名查找。

### 优化
- **UI 状态管理**：改进了战斗瞄准和资源追踪的状态处理。
- **资源限制**：在施法选择环阶时，系统现在会检查当前角色的剩余法术位，自动禁用已耗尽的环阶选项，防止超量使用。
- **自定义动作限制**：系统现在会尝试从角色的职业资源中读取"每轮动作" (Actions Per Turn) 等自定义字段，以支持拥有特殊规则的角色。
- **角色创建向导**：更新了 AI 系统提示词，现在会请求生成更详细的法术和特性数据（如学派、施法时间、射程、成分等）。
- **数据映射**：增强了角色创建完成后的数据处理逻辑，现在能够正确解析并保存法术和特性的详细属性到数据库中，提升了数据的完整性。
